# django博客

博客的初版前端是完全仿照 [panda 博主的博客](http://panda.panda-studio.cn/)来的。未来会进行一定的改版。

博客后端是django，初版是按照[追梦人物的博客](https://www.zmrenwu.com/)中的django教程进行编写的。
